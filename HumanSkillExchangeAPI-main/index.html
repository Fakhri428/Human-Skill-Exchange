HSE Human Skill Exchange API
Tujuan Alur Endpoint Tester RapiDoc

Praktikum Web Service

Human Skill Exchange API

REST API untuk mempertemukan pengguna berdasarkan skill, kebutuhan, penawaran, exchange request, progress, review, dan reputasi.

Buka Dokumentasi Coba Endpoint 

        

    






    

        

            
Bagian 1

            
Tujuan Project

        


        

            

                01
                
Profil dan Personal Branding

                
User dapat membuat profil, mencatat skill, dan menampilkan portofolio.

            


            

                02
                
Skill, Need, dan Offer

                
API mencatat kemampuan yang dimiliki, kebutuhan bantuan, dan penawaran kontribusi.

            


            

                03
                
Rule-Based Matching

                
Sistem memberi rekomendasi berdasarkan kategori, kata kunci, mode kerja, dan rating.

            


            

                04
                
Exchange dan Reputasi

                
User dapat mengirim request, update progress, menyelesaikan exchange, dan memberi review.

            

        

    


    

        

            
Bagian 2

            
Alur Sistem

        


        

            

                1
                

                    
Register dan Profil

                    
User register, login, lalu mengisi bio, lokasi, mode kerja, dan waktu tersedia.

                

            


            

                2
                

                    
Kontribusi dan Kebutuhan

                    
User menambahkan skill, need, dan offer sesuai kontribusi yang dapat ditukar.

                

            


            

                3
                

                    
Matching dan Request

                    
API menghitung kecocokan, lalu user mengirim exchange request kepada partner.

                

            


            

                4
                

                    
Progress, Review, Reputasi

                    
Exchange dijalankan, dikonfirmasi selesai oleh dua pihak, lalu dinilai dengan review.
